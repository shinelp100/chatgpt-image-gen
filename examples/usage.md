# 示例用法

## 示例 1：股票研报工作流

**输入**：
```
用 ChatGPT 生成图片

内容：股票研报工作流
流程：数据采集 → 数据分析 → 报告生成 → AI润色 → 发布

要求：
1、解析 Mermaid flowchart 语法，生成渐变背景 + 毛玻璃卡片的信息图
2、支持 TD（垂直）布局
3、自动水印（题材调研员）
4、主题：hand-drawn-edu
```
